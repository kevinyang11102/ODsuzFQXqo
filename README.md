# Python图书管理系统 python1202

## 项目概述

本项目是一个基于Python语言的图书管理系统，旨在帮助图书馆管理图书信息，提高图书管理效率。

## 技术栈

- 后端：Python 3.8+
- 前端：HTML, CSS, JavaScript
- 数据库：MySQL
- 框架：Flask

## 功能模块

### 用户模块

- 用户注册
- 用户登录
- 用户信息管理

### 图书模块

- 图书信息录入
- 图书信息查询
- 图书借阅
- 图书归还

### 管理模块

- 图书管理
- 用户管理
- 借阅记录管理

## 系统角色

- 普通用户：可以查看图书信息，借阅和归还图书。
- 管理员：拥有系统管理权限，可管理图书信息和用户。

## 运行环境

- 操作系统：Linux/Windows/MacOS
- Python版本：3.8+
- 数据库：MySQL 5.7+

## 部署步骤

1. 安装Python环境和依赖包
2. 安装MySQL数据库，并导入数据
3. 配置系统环境变量
4. 运行Flask应用

## 目录结构

```
project/
│
├── app/              # 应用目录
│   ├── models/       # 数据模型目录
│   ├── views/        # 视图层目录
│   ├── templates/    # HTML模板目录
│   └── static/       # 静态文件目录
│
├── config.py         # 配置文件
├── run.py            # 应用启动文件
└── requirements.txt  # 依赖包列表
```

- 界面简洁，易于操作。
- 基于Python和Flask，易于维护和扩展。
- 前后端分离，便于前端开发。
